API Documentation
1. Register

First you need to register to get a client_id which will be used for the API calls. Here

Keep the following in mind while registering :
  1. This service is completely free with no caps.
  2. The username you provide should be the same as your unsplash username
  3. The API currently only provides the Trending Images since it is unavailable on Unsplash API and I handselect n update those on daily basis.
  4. Since this service is directly used with Unsplash API hence you should only provide your genuine details.
  5. You will never receive any SPAM or other Emails from my end , be assured of that .
  6. This is my personal project and some API calls from my end use the Unsplash API and I am providing that from one of my other application's API limit . Hence it is a request that you don't abuse the API provided here .Also I am not earning anything from the same, just trying to help out the community :)
  7. Also since it is a personal project , please only send me queries for the same not Unsplash !!

2. Trending Images call

GET Request
https://api.dawnimpulse.com/wallup/unsplash_trending

Parameters
client_id        :  (important) The id received after registering
page              :  (optional) The page no you wish to return. (default : 1)
per_page      :  (optional) No of items per page. (default : 30)

Example :
https://api.dawnimpulse.com/wallup/unsplash_trending?client_id=abcabc&page=2&per_page=20

Response
Similar to Unsplash for success

Success will return an Array of Objects.

Error will return
{ "success" : "false" , errorMessage : "the error message" }
